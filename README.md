# The PySpark Field Guide

A visual, hands-on PySpark reference for SQL-fluent data engineers — from cluster basics to production pipelines.

🔗 **[Live Site →](https://abhi-rj02.github.io/pyspark-practical-reference/)**

## What's Inside

- **30 sections** across 5 parts — Foundations, DataFrames, Operations & Schemas, PySpark in Python, PySpark SQL & RDDs
- **11 worked examples** with real-world code you can run
- **SQL bridge callouts** mapping every PySpark operation to its SQL equivalent
- **Scenario callouts** — ETL pipelines, schema drift, warehouse migrations, executive reporting
- **Hard points** — common gotchas like silent type mismatches, position-based unions, driver OOM traps
- **Pro tips** — production advice for Databricks, performance tuning, and clean code patterns

## Topics Covered

| Part | Sections |
|------|----------|
| Foundations | Spark architecture, SparkSession, DataFrames, scalability & performance |
| DataFrames | Reading files, inspecting, aggregations, select/filter/sort, null handling |
| Schemas | CSV/JSON/Parquet, explicit schemas, StructType/StructField |
| PySpark in Python | Column ops, advanced transforms, unions, joins, UDFs, Pandas UDFs |
| SQL & RDDs | RDD basics, Spark SQL, temp views, complex aggregations, pivot/rollup/cube, broadcasting, end-to-end pipeline |
